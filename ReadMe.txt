介绍：
此程序主要用来通信加密，通过翻译代码来保护聊天记录。
使用方法：在对应位置输入字符，选填秘钥(英文)，按下“翻译”键来实现功能。
其优点在于为动态密文,每次生成都不同，故有极大的加密性。
注意：
请在中文输入法下此程序打开即可复制中文，改变字体时请确保其他栏为空。
备注：
如果安装到C盘，则每次访问时会弹出是否授予管理员权限的弹窗，故不建议安装到C盘
如果发生错误，请用文件夹中的"Debugger"文件尝试修复
支持的字体：
宋体, 楷体, 黑体, 微软雅黑, 仿宋, Consolas,
思源黑体, Romantic, 华文中宋, 华文仿宋, 华文宋体,
华文彩云, 华文新魏, 华文琥珀, 华文细黑, 华文行楷,
幼圆, 新宋体, 隶书, 等线, 方正姚体, 方正舒体,
Roboto, 华文隶书, Tahoma.
